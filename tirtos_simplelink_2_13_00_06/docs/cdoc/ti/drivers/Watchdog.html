typedef enum Watchdog_LibType {
    Watchdog_LibType_Instrumented,
    // instrumented
    Watchdog_LibType_NonInstrumented
    // non-instrumented
} Watchdog_LibType;
 
VALUES
LibType_Instrumented — The library supplied is prebuilt with logging and assertions enabled.
LibType_NonInstrumented — The library supplied is prebuilt with logging and assertions disabled.
DETAILS
This enumeration defines all the Watchdog library type provided by the product. You can select the library type by setting the Watchdog.libType configuration parameter.

Watchdog Library type

Configuration settings
Watchdog.libType = Watchdog.LibType Watchdog.LibType_Instrumented;
 
DETAILS
Deprecated: Please use ti.tirtos.TIRTOS.libType instead.
The Watchdog runtime is provided in the form of a library that is linked with your application. Several forms of this library are provided with the Watchdog product. This configuration parameter allows you to select the form of Watchdog to use.
The default value of libType is Watchdog_LibType_Instrumented. For a complete list of options and what they offer see LibType.
