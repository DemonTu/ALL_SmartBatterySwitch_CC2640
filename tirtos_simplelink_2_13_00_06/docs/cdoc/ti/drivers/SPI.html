C synopsis target-domain
typedef enum SPI_LibType {
    SPI_LibType_Instrumented,
    // instrumented
    SPI_LibType_NonInstrumented
    // non-instrumented
} SPI_LibType;
 
VALUES
LibType_Instrumented — The library supplied is prebuilt with logging and assertions enabled. Diags_USER1 mask logs basic information Diags_USER2 mask logs more detailed information
LibType_NonInstrumented — The library supplied is prebuilt with logging and assertions disabled.
DETAILS
This enumeration defines all the SPI library type provided by the product. You can select the library type by setting the SPI.libType configuration parameter.

DETAILS
Deprecated: Please use ti.tirtos.TIRTOS.libType instead.
The SPI runtime is provided in the form of a library that is linked with your application. Several forms of this library are provided with the SPI product. This configuration parameter allows you to select the form of SPI to use.
The default value of libType is SPI_LibType_Instrumented. For a complete list of options and what they offer see LibType.
